# Computação Gráfica

### Estrutura de Diretórios

O repositório está organizado de forma a separar os arquivos de código-fonte das imagens finais renderizadas:

* **`pov/`**: Diretório que contém os arquivos de script escritos em POV-Ray SDL. Estes arquivos são responsáveis por descrever matematicamente os objetos, definir a iluminação, aplicar as texturas e posicionar as câmeras das cenas em 3D.
* **`png/`**: Diretório destinado a armazenar os resultados visuais. Contém as imagens finais renderizadas (no formato `.png`) que foram geradas a partir do processamento e compilação dos scripts POV-Ray.

### Tecnologias Utilizadas

* **POV-Ray:** Ferramenta principal utilizada para a modelagem matemática, iluminação e renderização (traçado de raios) das cenas tridimensionais do projeto.
